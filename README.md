## ✨ Features

- ✅ **Task Management** – Add, edit, delete, complete tasks with ease.
- 🎤 **Voice Input** – Use Web Speech API to add tasks hands‑free.
- 🤖 **AI Categorization** – Automatically categorize tasks (Work, Personal, Shopping, Health, Education) using OpenRouter AI.
- 🌤️ **Live Weather** – Auto‑detects your location and shows current temperature, wind, humidity.
- 🕐 **Live Clock** – Real‑time clock with timezone support for your selected country.
- 📊 **Smart Statistics** – Track completion rate, streak, pending tasks with animated progress bars.
- ✅ **Subtasks / Checklists** – Break tasks into smaller steps with individual progress.
- 🔁 **Recurring Tasks** – Set daily, weekly, or monthly recurrence.
- 🎯 **Focus Mode** – Fullscreen focus timer for deep work.
- ⏱️ **Pomodoro Timer** – Built‑in 25/5 minute timer to boost productivity.
- 📅 **Calendar View** – See public holidays for 50+ countries with detailed popups.
- 📈 **Analytics Dashboard** – Weekly progress charts, category distribution, and stats.
- 🌙 **Dark / Light Mode** – Toggle themes, saved in local storage.
- 💾 **Export / Import** – Backup your tasks as JSON.
- 📲 **Progressive Web App** – Installable on mobile and desktop.

## 📦 Installation & Setup

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/manishrajdoot/smart-todo-pro.git
   cd smart-todo-pro
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**  
   Create a `.env` file in the root directory:
   ```env
   VITE_OPENROUTER_API_KEY=your_openrouter_api_key
   ```
   (Get a free API key from [OpenRouter](https://openrouter.ai/))

4. **Run the development server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

5. **Build for production**
   ```bash
   npm run build
   ```
   The output will be in the `dist` folder.

## 🌍 Deployment

### Deploy on Vercel (Recommended)

1. Push your code to a GitHub repository.
2. Go to [Vercel](https://vercel.com/) and import your repository.
3. Add environment variables (`VITE_OPENROUTER_API_KEY`).
4. Deploy – your app will be live in seconds.

### Deploy on Netlify / Render

Similar process – set build command to `npm run build` and publish directory to `dist`.
